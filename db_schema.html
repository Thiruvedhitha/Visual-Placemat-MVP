Access Control (RLS)

3 ways a user can access a capability_catalog:

1
Owner — catalog.user_id = auth.uid()
2
Shared — catalog_shares.user_id = auth.uid()
3
Client Member — client_members.user_id = auth.uid() AND catalog.client_id = client_members.client_id

Access cascades down:

capability_catalogs → capabilities → visual_maps → diff_history → prompt_sessions
